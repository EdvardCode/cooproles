# 🤝 Coop Roles (English)

Make Minecraft survival truly cooperative! **Coop Roles** is an asymmetrical co-op mod where every player becomes an irreplaceable part of the team. Surviving alone is no longer an option: you will have to communicate, share resources, and protect each other.

### 🎭 Available Classes:
* ⛏️ **Miner** — The only one who can break blocks and gather resources.
* 🧱 **Builder** — The only one who can place blocks and build the base.
* 🛠️ **Crafter** — The team's engineer. Only they can use crafting tables, furnaces, and anvils.
* ⚔️ **Warrior** — Your shield and sword. The only one capable of dealing damage to mobs.

### ⚙️ Mod Features:
* **HUD Overlay:** Your current role is always conveniently displayed in the top-left corner of the screen.
* **Persistent:** Roles are not lost after death or reconnecting to the server.
* **Gameplay:** The mod is disabled by default when creating a world. Gather some initial resources, assign roles, and run the enable command when you're ready for the hardcore experience!

### 💻 Commands (requires operator permissions):
* `/cooproles on` / `/cooproles off` — Enable or disable role restrictions.
* `/cooproles setrole <player> <role>` — Assign a role to a player (Tab-completion supported).
* `/cooproles role` — Check your current role (available to all players).

# 🤝 Coop Roles

Сделайте выживание в Minecraft по-настоящему командным! **Coop Roles** — это асимметричный кооперативный мод, где каждый игрок становится незаменимым звеном. Выжить в одиночку больше не получится: вам придется общаться, делиться ресурсами и защищать друг друга.

### 🎭 Доступные классы:
* ⛏️ **Добытчик (Miner)** — единственный, кто может ломать блоки и добывать ресурсы.
* 🧱 **Строитель (Builder)** — только он способен ставить блоки и возводить базу.
* 🛠️ **Ремесленник (Crafter)** — инженер команды. Только ему доступны верстаки, печи и наковальни.
* ⚔️ **Воин (Warrior)** — ваш щит и меч. Только он способен наносить урон мобам.

### ⚙️ Особенности мода:
* **HUD-интерфейс:** Ваша текущая роль всегда удобно отображается в левом верхнем углу экрана.
* **Надежность:** Роли не сбрасываются после смерти или перезахода на сервер.
* **Геймплей:** По умолчанию мод выключен при создании мира. Развейтесь, раздайте роли и пропишите команду запуска, когда будете готовы к хардкору!

### 💻 Команды (требуют права администратора):
* `/cooproles on` / `/cooproles off` — Включить или выключить ограничения ролей.
* `/cooproles setrole <игрок> <роль>` — Выдать роль игроку (работает автодополнение по Tab).
* `/cooproles role` — Узнать свою текущую роль (доступно всем).
